---------------------- PROYECTO INTEGRADOR -------------------------------

-- En que consiste el proyecto integrador --------------------------------

El proyecto integrador consiste en un el procesamiento de los datos de 
un archivo CSV(comma separated values) del cual se debe extraer la data
y colocarla dentro de un modelo de base de datos el cual se debe preparar
previamente, con esta extracción de la informacion se la procesa dentro
del modelo de base de datos utilizando el motor de base de datos MySQL
dentro del cual una vez ingresada la información, esta debe estar de una
forma en que permita la correcta forma de almacenar los datos y a la vez
que este modelo permita el rápido acceso a estos mismos.
